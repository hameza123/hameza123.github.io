Of course. Here is a comprehensive tutorial for setting up osTicket, a popular open-source help desk ticketing system, on your Windows Server 2022.

osTicket Setup Tutorial on Windows Server 2022 Link to heading

Prerequisites Link to heading

  • A Windows Server 2022 machine with internet access.
  • Administrator access to the server.
  • A web browser on the server.

Part 1: Install Web Server (IIS) and Application Request Routing (ARR) Link to heading

osTicket requires a web server (IIS) with PHP and MySQL support.

  1. Open Server Manager.

  2. Click Add roles and features.

  3. Click Next until you reach the “Server Roles” section.

  4. Expand Web Server (IIS) and select the following:

    • Web Server > Common HTTP Features > Default Document
    • Web Server > Common HTTP Features > Directory Browsing
    • Web Server > Common HTTP Features > HTTP Errors
    • Web Server > Common HTTP Features > Static Content
    • Web Server > Performance > Static Content Compression
    • Web Server > Application Development > CGI
    • Web Server > Application Development > ISAPI Extensions
    • Web Server > Application Development > ISAPI Filters

  5. Click Next and then Install. Wait for the installation to complete.

  6. Install Application Request Routing (ARR) for URL Rewrite:

    • Download and install the Microsoft URL Rewrite Module from the official Microsoft site. Search for “URL Rewrite IIS download”.
    • Download and install Application Request Routing from the official Microsoft site.

Part 2: Install PHP Link to heading

osTicket is written in PHP, so we need to install the PHP manager and runtime.

  1. Install PHP Manager for IIS:

    • Download the latest PHP Manager for IIS from the official GitHub page.
    • Run the downloaded .msi file to install it.

  2. Install PHP:

    • Download the non-thread-safe (NTS) x64 version of PHP 8.1 from windows.php.net/download. (osTicket v1.18+ supports PHP 8.1).
    • Create a folder for PHP on your system drive, e.g., C:\PHP.
    • Extract the contents of the downloaded ZIP file into this folder.

  3. Configure PHP in IIS:

    • Open Internet Information Services (IIS) Manager.
    • On the left, select your server.
    • Double-click PHP Manager.
    • Click on Register new PHP version.
    • Browse to the php-cgi.exe file inside your PHP folder (e.g., C:\PHP\php-cgi.exe) and click OK.

Part 3: Install MySQL (Database) Link to heading

osTicket needs a database to store tickets, users, and settings.

  1. Download MySQL Installer:

    • Go to dev.mysql.com/downloads/installer.
    • Download the MySQL Web Installer.

  2. Run the Installer:

    • Choose Server only as the setup type.
    • Follow the prompts. When you get to Accounts and Roles, set a strong root password. Remember this password!
    • Complete the installation.

Part 4: Install and Configure osTicket Link to heading

  1. Download osTicket:

    • Visit osticket.com/download and get the latest version (e.g., osTicket-v1.18.zip).

  2. Extract and Copy Files:

    • Extract the ZIP file.
    • Copy the upload folder to C:\inetpub\wwwroot.
    • Rename the folder from upload to osticket. Your path should now be C:\inetpub\wwwroot\osticket.

  3. Set Folder Permissions:

    • Navigate to C:\inetpub\wwwroot\osticket.
    • Right-click the osticket folder, select Properties > Security > Edit > Add.
    • Type IIS_IUSRS, click Check Names, then OK.
    • Give this group Modify permissions. Click OK.

Part 5: Final Configuration in IIS Link to heading

  1. Create a Site in IIS Manager:

    • In IIS Manager, right-click Sites > Add Website.
    • Site name: osTicket
    • Physical path: C:\inetpub\wwwroot\osticket
    • Binding: Leave the Type as http and Port as 80. You can also set a host name if you have one.
    • Click OK.

  2. Set Default Document:

    • In IIS, click on your new osTicket site.
    • Double-click Default Document.
    • Make sure index.php is in the list. If not, add it.

  3. Enable URL Rewrite:

    • Click on your osTicket site.
    • Open the URL Rewrite feature.
    • If there’s no rule, you may need to create one later based on osTicket’s requirements.

Part 6: Run the osTicket Web Installer Link to heading

  1. Open your browser on the server and go to http://localhost/osticket/.

  2. The osTicket installer screen will appear.

  3. Click Continue and follow the on-screen steps:

    • Database Settings:

      • MySQL Database: osticket
      • MySQL Username: root
      • MySQL Password: [The root password you set during MySQL installation]
      • MySQL Host: localhost

    • Site Settings:

      • Help Desk Name: Your Company Name
      • Default Email: The email address where help desk emails will be received.

  4. Complete the Installation. The installer will set up the database and create the configuration file.

  5. Delete the Setup Folder:

    • After successful installation, a critical security step is to delete the setup directory.
    • Navigate to C:\inetpub\wwwroot\osticket\setup\ and delete the entire setup folder.

Part 7: Access Your Help Desk Link to heading

  • Admin Panel: Go to http://localhost/osticket/scp and log in with the default admin credentials created during setup.
  • Client Portal: Go to http://localhost/osticket/ to submit new tickets.

Your osTicket help desk is now ready for use! You can further configure email settings, add agents, and customize workflows from the admin panel.
